How is the site organized?

The website is organized hierarchically. The homepage presents main sections such as Visit, Exhibits, Research, Education, and About, which act as top-level categories. Each of these sections contains subpages that provide more detailed information, allowing users to navigate from general topics to specific content easily. This structure makes it simple for visitors to find information without following a strict linear path, while still maintaining a clear and organized navigation system.

The website effectively uses the CRAP design principle of Contrast. For example, white text is used over the blue background to make the menu very visible which make navigation more easier The red "DONATE" button also provides a strong color contrast against the surrounding, white text and teal background,

The website is engaging. It uses visually appealing images, interactive virtual tours, and organized content to keep visitors interested. Make at least one recommendation to improve this website based on what you learned in this module.

The website could improve its accessibility by adding more descriptive alt text to images for screen readers. This would help visually impaired users better understand the content of the images and enhance their overall experience on the site.
