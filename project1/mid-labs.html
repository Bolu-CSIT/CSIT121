Overview

This page contains labs completed from the beginning of the semester up to the mid-semester point. These labs focused on building a strong foundation in HTML, basic CSS styling, and understanding proper document structure.

Included Labs

Lab 01 – HTML Structure

This lab focused on how to edit an HTML,preview it locally, validate it, and transfer it to a web server for public access. this lab was an introduction to the course and how to use VS code editor.

View Lab 01

Lab 02 – HTML Structure

This lab focused on learning the basic structure of an HTML document, including headings, paragraphs, lists, links, and semantic elements.

View Lab 02

Lab 03 – CSS Styling

In this lab, I applied external CSS stylesheets to improve layout, font styling, colors, and overall visual presentation of web pages. In this lab a CSS stylesheet was created to style the blog page in lab02

View Lab 03

Lab 04 – Visual Media

This lab is all about adding pictures to websites. A multi page website was created with different images on each page in different format including JPEG, PNG, and SVG, and CSS was used to style the background of each

View Lab 04

Lab 05 – Responsive Design

This lab introduced the use of Flexbox, grid and media queries and to create layouts that adapt to different screen sizes.
