In this lab, I learned how to design a blog page so it looks good on both mobile devices and desktop computers. I used tools like flexbox, grid, and media queries to make the design flexible and responsive to different screen sizes

Best Design Practices I Follow

  1. Keep layout simple and consistent across pages
  2. Use responsive design for both mobile and desktop users
  3. Choose readable fonts and accessible color contrasts
